# odin-recipes
The first project in 'The Odin Project' foundations course. It contains 3 recipes of food that I like. 

This project was built purely using HTML. Nothing was particularly challenging
as I have previous experience with HTML. I had a refresher of html tags and
elements such as the anchor and the two lists. Honestly, the most difficult part
of this project was the copying and pasting of ingredients and steps!

Hoping for more fun along the way.
